Welcome to ECS Parser!

Developed by SkyFuInMC, this web app is used for deliver news in a secure way.

All information is maintained in the given link, so you will not change anything inside the article unless you changed the link.

How to use it?

All editing action is done by clicking “Edit” button!

It's easy, right?

Create an article

First, you can write down your name (or any name you want though) in the Author Box. Then write your article in HTML inside Content box!

At last, click Submit button to generate a link!

Yay, you done it!

PS: You may use a URL shorten service to shorten the link.

How to style it?

It's same as you edit an HTML file, all markups are in HTML.
